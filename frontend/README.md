# Smart Health Assistant Frontend

This is a minimal React frontend for the Multi-Modal Early Disease Screening Platform.

## Features
- Symptom Checker
- Risk Assessment
- Image Upload

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm start
   ```
3. The app will run at `http://localhost:3000`.
